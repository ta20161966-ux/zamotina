# Currency Converter

**Автор:** Иванов Иван

## Получение API-ключа
1. Зарегистрируйтесь на exchangerate-api.com.
2. Получите ключ в личном кабинете.
3. Вставьте ключ в код приложения.

## Запуск
1. Установите зависимости: `pip install requests`
2. Запустите: `python main.py`

## Примеры использования
- Введите сумму и выберите валюты.
- Нажмите «Конвертировать».
- История сохраняется в `history.json`.
